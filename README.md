# AI-Powered Job Vacancy Portal

A modern, full-stack web application designed for job seekers and recruiters, featuring AI-powered resume parsing and job matching recommendations.

## Total Architecture
- **Frontend**: Next.js 15 (React), pure CSS Modules (SaaS aesthetic, dark mode styling).
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose models).
- **Authentication**: Custom JWT authentication and Role-based Access Control (Seeker, Recruiter, Admin).

## Features
- **Job Seekers**: AI Smart Matching, Job Searching, Application Tracking, Resume Data Parsing.
- **Recruiters**: Job Posting, Applicant Tracking, Dashboard Analytics.
- **Modern UI**: Full Dark Mode, gradient themes, glassmorphism UI.

## Setup Instructions

### Prerequisites
- Node.js v18+
- MongoDB Local Server (`mongodb://127.0.0.1:27017`) or cluster URL.

### 1. Starting the Backend
1. Open a terminal and navigate to the `backend` folder:
   ```bash
   cd backend
   ```
2. Install dependencies (already installed if generated by AI):
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   *The server will run on http://localhost:5000*

### 2. Starting the Frontend
1. Open a new terminal and navigate to the `frontend` folder:
   ```bash
   cd frontend
   ```
2. Run the Next.js development server:
   ```bash
   npm run dev
   ```
   *The client will run on http://localhost:3000*

## Sample Test Data (API Postman Testing)

**Register a Recruiter:**
`POST http://localhost:5000/api/auth/register`
```json
{
  "name": "Recruiter Jane",
  "email": "jane@techcorp.com",
  "password": "password123",
  "role": "recruiter"
}
```

**Post a Job (Requires Recruiter Token):**
`POST http://localhost:5000/api/jobs`
```json
{
  "title": "AI Engineer",
  "company": "TechCorp",
  "location": "Remote",
  "salary": "$150k - $200k",
  "description": "Build cutting-edge AI models.",
  "requirements": ["Python", "TensorFlow", "React"]
}
```

**Get Mock AI Resume Parsing:**
`POST http://localhost:5000/api/ai/parse-resume` (With token auth)
Returns an extracted `skills` array.
